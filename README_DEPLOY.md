# MLN111 Quiz App v10

Bản v10 đã bổ sung mô tả thao tác trực tiếp trên màn hình và rà lỗi tiếng Việt/diacritics trong bộ câu hỏi.

## Phím tắt

- `1-9` hoặc `A-E`: chọn đáp án.
- `Enter` / `Space`: qua câu tiếp theo sau khi đã trả lời. Trong Flashcard, dùng để lật thẻ.
- `←` / `→`: lùi hoặc tiến câu.
- `R`: chọn lại.
- `F`: đánh dấu câu cần ôn.
- `M`: đánh dấu đã thuộc.
- `/`: focus vào ô tìm kiếm.

## Deploy Vercel

- Framework Preset: `Other`
- Build Command: `npm run build`
- Output Directory: `dist`

Tiến độ lưu bằng `localStorage`, không cần backend/database.
